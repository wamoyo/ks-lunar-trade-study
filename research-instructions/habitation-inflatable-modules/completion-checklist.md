## Specification Coverage Checklist

**For each item in the JSON file, verify:**

### Critical Specifications (Must Have Source or "Unavailable" Note)
- [ ] **TRL:** Verified from NASA, peer-reviewed source, or marked "unverified"
- [ ] **Vendor Name:** Official company name confirmed
- [ ] **Model/Variant:** Specific designation or project name documented
- [ ] **Application:** Clearly marked as "orbital", "surface", or "dual-use"

### High-Priority Specifications (Document if Available)
- [ ] **Mass (kg):** Dry mass, wet mass, or deployed mass with source
- [ ] **Volume (m³):** Stowed and/or deployed volume with source
- [ ] **Crew Capacity:** Number of crew and duration with source
- [ ] **Physical Dimensions:** Length, diameter, height with source

### Medium-Priority Specifications (Document if Available)
- [ ] **Power (kW):** Total power consumption with source
- [ ] **Cost (USD):** Estimated or actual cost with source
- [ ] **Turnaround Time:** Order-to-delivery timeline with source
- [ ] **Design Life:** Intended operational lifespan with source

### Lower-Priority Specifications (Document if Available)
- [ ] **Radiation Protection:** Dose reduction or shielding specs
- [ ] **Thermal Range:** Operating temperature range
- [ ] **Micrometeorite Protection:** Impact resistance specifications
- [ ] **Dust Mitigation:** Lunar regolith management approach
- [ ] **Launch Vehicle Compatibility:** Which rockets can carry it
- [ ] **Assembly Requirements:** Crew time or robotic assistance needed

## Quality Assurance Checklist

### Source Credibility
- [ ] Every data point has source URL or explicit "unavailable" note
- [ ] TRL values only from Tier 1 sources (NASA TechPort, NASA contracts) or marked "unverified"
- [ ] No specifications from Tier 5 sources (blogs, forums) without verification
- [ ] Conflicting specifications flagged for Phase 4 verification

### Data Completeness
- [ ] All TRL 5+ systems included (none excluded due to obscurity)
- [ ] International vendors represented (not only US companies)
- [ ] Small companies and startups included (not only primes)
- [ ] Both mature (TRL 8-9) and emerging (TRL 5-6) technologies included

### Data Accuracy
- [ ] No estimated or calculated values without explicit source
- [ ] No extrapolations from one specification to another without source
- [ ] Units clearly specified (kg not lbs, m³ not ft³, kW not W)
- [ ] Dates noted for time-sensitive data (e.g., "TRL 6 as of 2024")

### Categorization
- [ ] Clear distinction between orbital and surface habitats
- [ ] Clear distinction between complete systems and components/subsystems
- [ ] Clear distinction between operational (TRL 9) and developmental (TRL 5-8)
- [ ] Hybrid systems appropriately categorized
