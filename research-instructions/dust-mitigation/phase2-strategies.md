# Phase 2: Search Strategies - Dust Mitigation

**Date:** 2025-12-08
**Category:** Dust Mitigation Systems for Lunar Operations

## Overview

Based on Phase 1 findings, lunar dust mitigation is a specialized, NASA-dominated field with limited commercial availability. Most technologies are at TRL 3-6, with only 4 of 31 identified methods at TRL 7+. This phase defines comprehensive search strategies to identify all TRL 5+ systems and vendors.

## Search Strategy Framework

### Strategy 1: NASA TechPort Comprehensive Search
**Objective:** Identify all NASA-funded dust mitigation projects at TRL 5+

**Approach:**
1. Search TechPort web interface with multiple keyword combinations:
   - "lunar dust mitigation"
   - "dust removal"
   - "electrodynamic dust shield"
   - "electrostatic dust"
   - "dust coating"
   - "dust cleaning"
   - "regolith mitigation"
   - "contamination control lunar"

2. Filter by:
   - TRL: 5, 6, 7, 8, 9
   - Technology Areas: TX06.3 (Atmospheric Control and Monitoring), TX06.4 (Habitation Systems)
   - Status: Active, Completed (last 5 years)

3. For each project found:
   - Record project ID
   - Extract organization/vendor name
   - Note TRL start/current/end
   - Capture technical specifications
   - Identify PI contact information
   - Check for related projects

**Expected Coverage:** 10-20 projects

**Specific Projects Identified in Phase 1:**
- 32733 - Electrodynamic Dust Shield for Lunar/ISS Experiment
- 102790 - PolyK Dust Mitigation Laminate
- 146445 - Electrodynamic Dust Shield for Active...
- 113118 - Bendable Electrodynamic Dust Shields
- 117455 - Dust Solution Testing Initiative (DuSTI)
- 106645 - Dust In-situ Manipulation System (DIMS)
- 113369 - Modal Optimized Vibration dust Eliminator (MOVE)

### Strategy 2: NASA SBIR/STTR Database Mining
**Objective:** Track companies that received dust mitigation funding and identify commercial products

**Approach:**
1. Search NASA SBIR database (sbir.nasa.gov) for:
   - Topic: "Lunar Dust Mitigation Technology for Spacesuits"
   - Topic: "Active and Passive Dust Mitigation Surfaces"
   - Topic: "xEMU Lunar Dust Mitigation Devices"
   - Topic: "Liquid Crystal Membrane Dust Mitigation System"

2. Filter by:
   - Phase II awards (closer to commercialization)
   - Awards from 2020-2025 (recent/relevant)
   - Contract completion status

3. For each company:
   - Check company website for commercial products
   - Search for follow-on funding (Phase III, other contracts)
   - Verify current business status (active/acquired/defunct)
   - Contact information for product availability
   - Look for press releases about product launches

4. Cross-reference with:
   - SAM.gov contract database
   - NASA procurement notices
   - Company LinkedIn/Crunchbase profiles

**Expected Coverage:** 15-30 companies

## Data Collection Requirements

For each technology/system identified:

### Required Fields:
- **Vendor/Organization:** Name, type (company/university/NASA center)
- **Model/Technology Name:** Specific designation
- **TRL:** Current level with source
- **Category:** Active/Passive, technology type
- **Application:** What it protects (suits, optics, solar, etc.)
- **Mass (kg):** If available
- **Power (kW):** If available
- **Performance:** Removal efficiency, effectiveness metrics
- **Cost (USD):** Estimated if available
- **Turnaround Time:** When can it be acquired?
- **Status:** Development, testing, operational, commercial
- **Contact:** Website, email, PI name
- **Notes:** Limitations, requirements, unique features

### Source Requirements:
- Every data point must have source URL
- Mark verified: false initially
- Add verification_date in Phase 4
- Capture screenshots for key specs

## Risk Mitigation

### Potential Issues:

**Issue:** TechPort web interface doesn't provide full data
**Mitigation:** Use API documentation, contact TechPort support, NTRS papers

**Issue:** SBIR companies no longer active
**Mitigation:** Check business registries, LinkedIn, Crunchbase, acquisition records

**Issue:** Limited public technical specifications
**Mitigation:** Contact PIs/companies directly, check conference papers, FOIA requests

**Issue:** TRL ratings inconsistent or unclear
**Mitigation:** Use NASA TRL handbook definitions, note uncertainty, cite source

**Issue:** International sources in non-English
**Mitigation:** Use translation tools, focus on English abstracts, ESA sources in English
